Setting up

For the language server project we are going to need a few things:

  • The Language Server Protocol Spec in our hand open in the browser (or epaper, good choice)

  • A language to do the project in (there will be a list of target ones below)

For me i went with Rust for the language server, but still checking out code examples in another languages if i didn't get entirely satisfied
